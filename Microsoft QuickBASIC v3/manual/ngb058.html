CSRLIN                   Line Position of Cursor

 y = CSRLIN

    Returns the row number of the current cursor location on the current
    screen page.

   ----------------------------------------------------------------------

      Notes:    To get the current column position, use POS. To set the
                cursor position, use LOCATE.
See Also: LOCATE POS
