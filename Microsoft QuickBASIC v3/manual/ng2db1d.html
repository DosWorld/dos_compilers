SGN                      Sign of Number

 y = SGN(numexpr)

    Returns the sign of its numeric argument.

     numexpr    A numeric expression.

   ----------------------------------------------------------------------

      Notes:    If numexpr is positive, SGN(numexpr) returns 1.

                If numexpr is 0, SGN(numexpr) returns 0.

                If numexpr is negative, SGN(numexpr) returns -1.
