ASC                      ASCII Value of First Character

 y = ASC(stringexpr)

    Returns the ASCII value of the first character of a string expression.

  stringexpr    A string expression.

   -----------------------------------------------------------------------

      Notes:    If the argument is a null string, ASC returns an Illegal
                Function Call error.
