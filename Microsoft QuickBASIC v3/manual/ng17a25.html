LET                      Assignment

 [LET] variable = expression

    Assigns the value of an expression to a variable.

    variable    A valid variable name.

  expression    The value assigned to variable. expression and variable
                must match in type.

   ----------------------------------------------------------------------

      Notes:    The keyword LET is optional.
