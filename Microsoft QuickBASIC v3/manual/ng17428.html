KILL                     Delete File(s)

 KILL filespec

    Deletes one or more files from disk.

    filespec    A string expression that follows DOS file naming
                conventions.

   -----------------------------------------------------------------------

      Notes:    Currently open files may not be deleted via KILL.

                This command is equivalent to DOS's DEL (ERASE) command.
